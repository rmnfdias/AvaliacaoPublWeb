O que é Publicação Web?

É o processo de disponibilizar conteúdo (textos, imagens, vídeos, etc.) ou aplicações acessíveis na internet. Isso é feito usando servidores web, hospedagem e um domínio.

Existem dois tipos de aplicações web, são elas as aplicações estáticas e as aplicações dinâmicas

Aplicações estáticas:

São compostas por arquivos HTML, CSS e JavaScript que não mudam dinamicamente. São rápidas e simples de criar, mas não interagem com bancos de dados.

Aplicações dinâmicas:

Interagem com bancos de dados e podem exibir conteúdo personalizado para cada usuário. Por exemplo, redes sociais e e-commerces.

E quais são suas diferenças?

Geralmente paginas estáticas costumam ser mais simples, carregam com mais facilidade e são mais faceis de serem hospedadas, já as paginas dinâmicas costmam oferecer funcionalidades mais avançadas, em contra partida exigem maiores recursos do servidor onde estarão hospedadas.

Domínio e DNS

Domínio:

É o endereço amigável usado para acessar um site (ex.: www.google.com; www.g1.com.br; www.github.com.br). Ele substitui o IP do servidor, tornando a navegação mais fácil e acessível.

DNS (Domain Name System):

É o sistema que traduz o domínio para o endereço IP do servidor onde o site está hospedado. Exemplo: Traduz www.google.com para algo como 142.250.190.46.

Servidores Compartilhados e Dedicados

Servidor compartilhado:

Trata-se de um único servidor que é utilizado por vários usuários, trazendo a vantagem da redução do custo para operar no servidor, além de ter uma fácil configuração.

A desvantagem dos serviores compartilhados são os recursos que na maioria das vezes são limitados, possui uma menor segurança e por ter acessos simultâneos o desempenho acaba sendo prejudicado.

Servidor dedicado:

O servidor é utilizado exclusivamente por um cliente, seu alto desempenho, segurança e controle

A desvantagens destes servidores são os custo elevados já que é necessário de conhecimentos técnicos para gerenciar.

Servidores Web

É o ambiente em que os arquivos de um site são armazenados e disponibilizados na internet. É ele quem recebe, armazena processa e posteriormente fornece as infomações buscadas na internet pelo usuário.

Os dois principais servidores são Apache e Nginx

Apache:

O servidor web mais antigo e utilizado na internet, é conhecido pela flexibilidade e compatibilidade com vários sistemas operacionais e tecnologias. É gratuito, de código aberto e multiplataforma.

Nginx

Um servidor web mais recente, lançado em 2004, é conhecido pela escalabilidade, velocidade e eficiência. É gratuito, de código aberto e multiplataforma. É muito utilizado em sites de grande escala.

Banco de Dados

Um servidor de banco de dados é um sistema informático que armazena, organiza, protege e fornece acesso a um ou mais bancos de dados. Ele pode ser um hardware dedicado ou um servidor virtualizado.

Serviço SSH

O SSH (Secure Shell) é um protocolo de segurança que permite a transferência de dados entre um computador e um servidor remoto de forma segura. Ele funciona como uma "ponte segura" entre o usuário e o servidor, criptografando toda a comunicação e protegendo-a contra ameaças externas.

Linux e Comandos Básicos

O Linux é um sistema operacional para computadores desenvolvido a partir de um modelo de código aberto. le é conhecido por sua flexibilidade, capacidade de personalização e eficiência, e é uma alternativa popular aos principais sistemas operacionais pago

Principais distribuições para servidores:

Ubuntu Server: Amigável para iniciantes.

CentOS (agora AlmaLinux): Estável e usado em empresas.

Debian: Focado em estabilidade.

Red Hat Enterprise Linux (RHEL): Suporte corporativo.
